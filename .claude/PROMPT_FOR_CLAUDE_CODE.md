# Demo's Cafe デモサイト 構築プロンプト

## ディレクトリ
このファイルが置かれているディレクトリ（`demos-cafe/`）内にすべて構築すること。

```
demos-cafe/
├── index.html          # メインページ（単一HTMLファイル）
├── css/
│   └── style.css       # スタイルシート
├── js/
│   └── main.js         # スクロールアニメーション等
└── assets/
    └── images/         # プレースホルダー画像（後から差し替え前提）
```

---

## サイト概要

**ブランド名**: Demo's Cafe（デモズカフェ）  
**業態**: 台湾スイーツ・タピオカドリンク・パンケーキのカフェチェーン  
**コンセプトコピー**: "DEMO'S FILL YOUR BODY WITH JOY."  
**ターゲット**: 20〜30代女性、SNS映えを重視するカフェ好き  
**目的**: ブランドの世界観を伝え、商品・店舗情報へ誘導するLPサイト

---

## デザイン仕様

### カラーパレット（厳守）

| 役割 | 変数名 | HEX |
|------|--------|-----|
| メインアクセント | `--color-mint` | `#3DBFA0` |
| ディープミント | `--color-mint-deep` | `#2A8F78` |
| ミストミント（薄） | `--color-mint-mist` | `#D6F5EE` |
| ベース | `--color-white` | `#FFFFFF` |
| テキスト | `--color-ink` | `#1A1A1A` |
| テキストサブ | `--color-ink-muted` | `#555555` |
| 背景グレー | `--color-bg-subtle` | `#F7F7F5` |

- **イエローは一切使わない**（元サイトの配色をミントグリーンに完全置換）
- ミントグリーンのみをアクセントに使う「モノアクセント戦略」を維持する
- グラデーションは使わない、フラットな配色で統一

### タイポグラフィ

- **英字ディスプレイ（見出し・キャッチ）**: `'Playfair Display'` または `'Bebas Neue'`（Google Fonts）— 太字・大きめ
- **商品名（手書き風）**: `'Dancing Script'`（Google Fonts）— ミントグリーンで表示
- **日本語本文**: `'Noto Sans JP'`（Google Fonts）— weight 400、細め
- **セクションタイトル**: 全大文字・広めのレタースペーシング（`letter-spacing: 0.2em`）

### レイアウト原則

1. **ヒーローセクション**: 全画面幅の食品写真（プレースホルダーOK）＋キャッチコピー大書き
2. **商品セクション**: 各商品ブロックは全幅写真＋手書き風の商品名ラベル（ミントグリーン）
3. **「はみ出し矩形」演出**: 商品写真の背後にミントグリーンの四角形をずらして配置（`position: absolute` で実装）
4. **余白重視**: セクション間は `padding: 80px 0` 以上、テキスト周辺も余裕を持たせる
5. **CTAボタン**: ミントグリーン背景＋白テキスト、または薄ミント背景＋ディープミントテキスト

## 実装するセクション（上から順に）

### 1. ナビゲーション
- ロゴ（テキストで代替可）左配置
- 右側に `TOP / BRAND MENU / SHOP / ABOUT / CONTACT` のリンク
- スクロールで背景が白になるJS実装（初期は透明）

### 2. ヒーロー（`#hero`）
- 全幅・100vh
- 背景: ダークなオーバーレイ付き食品プレースホルダー画像（CSS で `background-color: #2A8F78` で代替可）
- キャッチコピー: `DEMO'S FILL YOUR BODY WITH JOY.` を白・大文字・太字で左配置
- サブコピー: `カラダもこころも、満たして欲しい。`（日本語・小さめ）

### 3. コンセプト（`#concept`）
- 中央揃え、余白多め
- セクションタイトル: `CONCEPT`
- 3〜4行の日本語テキスト（ブランドの想いを語る文章）
- ミントグリーンのアクセントラインを見出し下に配置

### 4. プロダクト（`#product`）
以下の4商品を順番に配置。交互に左右を入れ替える（奇数: 写真左・テキスト右 / 偶数: 写真右・テキスト左）。

各商品ブロックの構成：
- 全幅写真エリア（プレースホルダー画像 or `background-color` で代用）
- 手書き風英字の商品名（`Dancing Script`、ミントグリーン、大きめ）
- 商品の日本語キャッチコピー（1〜2行）
- 商品説明文（2〜3行）
- ミント矩形の「はみ出し」演出を各ブロックに適用

商品リスト：
| 商品名（英） | 商品名（日） | 説明文 |
|-------------|------------|--------|
| Pancake | ふわしゅわ台湾パンケーキ | しっとりとした生地にたっぷりの生クリームをのせた、台湾発のふわしゅわパンケーキ。 |
| Tapioca | Demo's 自家製タピオカ | 職人が毎日仕込む自家製タピオカ。もちもちの食感と豊富なフレーバーが自慢。 |
| Pandora | パンドラ | もっちり生地にたっぷりクリームを詰めた、Demo's Cafe名物のオリジナルスイーツ。 |
| Honey Lemon | レウレウハニーレモン | 爽やかな酸味と蜂蜜の甘さが絶妙なバランスのドリンク。見た目も映えると大人気。 |

### 5. メニューCTA（`#menu-cta`）
- ミントミスト（`#D6F5EE`）背景
- `おいしいモノ、たくさん。` の日本語見出し
- `MENU →` ボタン（ミントグリーン）

### 6. ショップ（`#shop`）
- セクションタイトル: `SHOP`
- 店舗写真プレースホルダー（全幅）
- `SHOP LIST →` ボタン

### 7. FC・コンタクト（`#fc` `#contact`）
- 2カラムグリッド
- 左: FC加盟の案内テキスト＋ロゴ
- 右: CONTACT ＋ `MORE →` ボタン

### 8. フッター
- ナビリンク再掲（小さめ）
- ロゴ
- コピーライト: `© Demo's Cafe All Rights Reserved.`

## アニメーション要件（JS）

- スクロール時にセクションがフェードインで出現（`IntersectionObserver` 使用）
- 商品ブロックのミント矩形は少し遅延してスライドイン
- ナビは `scrollY > 50` でホワイト背景に切り替え（`transition: background 0.3s`）
- スマホ対応: ハンバーガーメニュー実装

---

## レスポンシブ対応

- ブレークポイント: `768px`（スマホ）、`1200px`（PC）
- スマホでは商品ブロックが縦並びに変わる
- ナビはハンバーガーメニューに切り替え

---

## 画像プレースホルダーの扱い

実際の画像がないため、以下で代替する：
- `background-color` にミントグリーン系のトーン（`#2A8F78` や `#3DBFA0`）を使う
- `aspect-ratio: 4/3` または `16/9` でエリアを確保
- 中央に `[商品名 Photo]` のテキストを白で表示
- 後から `<img>` タグに差し替えやすいよう、クラス名を `product-image` で統一

## 注意事項

- 外部ライブラリは Google Fonts のみ使用（CSSファイル内で `@import`）
- JS フレームワーク不使用（バニラJS のみ）
- `index.html` / `css/style.css` / `js/main.js` の3ファイル構成を守る
- CSS変数（カスタムプロパティ）を `:root` に定義してから使う
- コード内コメントは日本語で記述
